# Kanban Service Backend

Kanban system for managing products

## Старт сервиса

1. Клонируем репозиторий проекта:
```
git clone https://github.com/LinqCod/kanban-backend.git
```
2. Дальше необходимо запустить само приложение. Сервис обернут в докер контейнер. Для развертывания использовался `docker-compose`. Помимо контейнера сервиса поднимается также контейнер с postgres.

### Команды

Для запуска сервиса достаточно использовать команду `make service-up`, которая сбилдит и запустит приложение, доступное на порту 8080:

```
make service-up
```

Для завершения работы сервиса следует воспользоваться командой:

```
make service-down
```

## Эндпоинты

* GET /api/v1/products - Возвращает массив JSON-объектов продуктов
